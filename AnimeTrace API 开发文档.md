# AnimeTrace API 开发文档

（最后更新：2025年8月1日）

## 识别接口

**请求方式及地址：**
POST `https://api.animetrace.com/v1/search`

---

## 接口说明

| 参数 | 解释 | 可选参数 | 请求方式 | 是否必须 |
| --- | --- | --- | --- | --- |
| `is_multi` | 是否显示多个结果 | `0` 或 `1` | POST | 否 |
| `model` | 使用的识别模型（模型选择见下表） | 见下表（模型名）一列 | POST | 否 |
| `ai_detect` | 是否开启AI图检测 | `1`: 是 <br>`2`: 否 | POST | 否 |
| `file` | 识别的图片数据 (可选) | [图片数据] | POST | 是* |
| `url` | 识别的图片URL来源 (可选) | URL | POST | 是* |
| `base64` | 识别的图片Base64 (可选) | Base64 str | POST | 是* |

> **注意：** `file`、`url`、`base64` 三种传参方式选择其中一种作为图片数据传入即可，但必须提供一种。

## 状态码说明

| 状态码 | HTTP 状态码 | 中文信息 | 日文信息 |
| --- | --- | --- | --- |
| `17720` | - | 识别成功 | 認識成功 |
| `200` | 200 | Success | 成功 |
| `17721` | 200 | 服务器正常运行中 | サーバーは正常に稼働しています |
| `17701` | 413 | 图片大小过大 | 画像サイズが大きすぎます |
| `17702` | 503 | 服务器繁忙，请重试 | サーバーが忙しいです。もう一度お試しください |
| `17703` | - | 请求参数不正确 | リクエストパラメータが正しくありません |
| `17704` | 403 | API 维护中 | APIはメンテナンス中です |
| `17705` | - | 图片格式不支持 | サポートされていない画像形式です |
| `17706` | - | 识别无法完成（内部错误，请重试） | 認識に失敗しました（内部エラー、もう一度お試しください） |
| `17707` | - | 内部错误 | 内部エラー |
| `17708` | - | 图片中的人物数量超过限制 | 画像内の人物数が制限を超えています |
| `17722` | - | 图片下载失败 | 画像のダウンロードに失敗しました |
| `17728` | - | 已达到本次使用上限 | 今回の使用上限に達しました |
| `17731` | - | 服务利用人数过多，请重新尝试 | サーバーリソースが多すぎます。もう一度お試しください |
| `404` | - | 页面不存在 | ページが見つかりません |
